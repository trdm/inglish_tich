Из словаря Apresyan (En-Ru)

manager

['mænıdʒə] n

1. 1) управляющий, заведующий, руководитель, администратор; директор; менеджер

canteen manager - заведующий или директор столовой

sales manager - заведующий отделом сбыта

2) юр. управляющий имуществом

2. хозяин

good [bad] manager - хороший [плохой] хозяин

my wife is an excellent manager - моя жена - отличная хозяйка

she is not much of a manager - она не очень хорошая хозяйка

3. импресарио

4. парл. представитель одной из палат, уполномоченный вести переговоры по вопросу, касающемуся обеих палат

5. амер. деятель политической партии, занимающийся, в основном, организационными вопросами
